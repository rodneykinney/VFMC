If you get a "Could not download. Virus detected" message when downloading:
Go to "Windows Security > Virus and Thread Protection Settings > Manage Settings > Real-time protection > Off"

If you get a "Windows protected your PC" dialog:
Click "More info > Run anyway"

If you get a error "No such file or directory: C:\\Users\\...\\vfmc.log" error:
Manually create the expected directory and try again.
